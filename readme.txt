== Installation ==

***From your WordPress dashboard***
 1. Visit 'Plugins > Add New'
 2. Search for `wp-optigration`  or upload zip file
 3. Activate `wp-optigration` from your Plugins page

***Manual Installation***
 1. [Download](https://wordpress.org/plugins/wp-optigration/) "wp-optigration".
 2. Upload the `wp-optigration` directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
 3. Activate `wp-optigration` from your Plugins page.
